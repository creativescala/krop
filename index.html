Krop

Krop is a Scala 3 web framework. It's can handle any web application, from APIs to rich user experiences, but it excels at projects where productivity and feature richness are a priority. For more, see our principles.

Getting Started

The fastest way to start with Krop is using the template project, which is described in the Quick Start.

If you want to add Krop to an existing project, add the following to your build.sbt

libraryDependencies += "org.creativescala" %% "krop-core" % "0.7.0"
